# Google Search Console: Post Deployment Steps

Run these steps after each SEO deployment.

1. Open Google Search Console.
2. Add or verify the property for https://www.requityapp.com.
3. Submit the sitemap: https://www.requityapp.com/sitemap.xml
4. Use URL Inspection to request indexing for the homepage and key resource pages:
   - https://www.requityapp.com/
   - https://www.requityapp.com/resources.html
   - https://www.requityapp.com/how-to-find-a-good-realtor.html
   - https://www.requityapp.com/where-to-find-good-real-estate-agents.html
   - https://www.requityapp.com/find-a-real-estate-agent.html
   - https://www.requityapp.com/real-estate-agent-matching.html
   - https://www.requityapp.com/real-estate-agent-faq.html
   - https://www.requityapp.com/questions-to-ask-a-real-estate-agent.html
   - https://www.requityapp.com/buyers/find-buyers-agent.html
   - https://www.requityapp.com/sellers/find-listing-agent.html
   - https://www.requityapp.com/how-it-works.html
   (URL Inspection has a daily quota; spread the remaining resource pages over
   the following days, or rely on the sitemap for the rest.)
5. Inspect https://www.requityapp.com/auth. If it still appears in search
   results after the 301 redirect deploys, use Removals (temporary removal)
   or request reindexing of /auth so Google picks up the redirect and drops it.
6. Wait for crawl and ranking changes. This can take days or weeks. Monitor
   Pages (indexing) and Performance reports for the new resource pages.

Notes:

- /auth 301 redirects to /agent/login.html (configured in vercel.json).
- Non-www requityapp.com 301 redirects to https://www.requityapp.com.
- robots.txt disallows /auth, /agent/dashboard.html, /reviewer/, /api/, and
  /seo/ (internal planning files such as realtor-question-map.md).
- Private pages (agent dashboard, reviewer pages) carry noindex, nofollow.
- Public resource pages are generated by scripts/generate-seo-pages.mjs from
  content data in scripts/seo-content/. Re-run the script after content edits
  and update sitemap lastmod dates.
