All market landing page skyline images were sourced from Wikimedia Commons.
Every image is public domain, CC0, or a Creative Commons license that allows
commercial use. Licenses were verified on the Commons file page at download
time.

Attribution is kept in this document. None of these licenses require visible
attribution on the landing page itself; CC BY and CC BY-SA require attribution
"in a reasonable manner", which this document provides. If an image is ever
swapped, update its row here.
